## Usage
To use this rule either add it to your `reflex.yaml` configuration file:  
```

```

or add it directly to your Terraform:  
```
...

module "reflex-aws-sqs-public" {
  source           = "github.com/reflexivesecurity/reflex-aws-sqs-public"
}

...
```
